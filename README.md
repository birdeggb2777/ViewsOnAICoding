# Views On AI Coding

本頁紀錄TIOBE排名前20名的程式語言，這些程式語言的創造者，他們對AI coding的看法。

以下連結僅列出第一手來源，有興趣者可以直接從連結過去看，或尋找更多資訊，**切勿**只看以下片段而做判定，避免斷章取義。

為了避免資料過時，列出了連結最後更新的日期。

**(若無特別說明，以下內容不包含我的意見或評論)**

---

[1] Python之父 (Guido van Rossum)
- 連結更新於：2025年11月25日
- 看法：**看法正面**，他預期人工智慧之後能夠學會使用任何對人類有效的語言。
- 連結：https://www.youtube.com/watch?v=QXE5rEVlu20

[3] Java之父 (James Gosling)
- 連結更新於：2026年02月左右
- 看法：**嚴正反對**，他表示這是竊盜行為，擁有原創性的才是最後的贏家。
- 連結：https://www.linkedin.com/posts/jamesgosling_the-biggest-ai-coverup-just-got-exposed-activity-7416903997854273536-MekC

[4] C++之父 (Bjarne Stroustrup)
- 連結更新於：2026年05月左右
- 看法：**不看好**，他認為會產生更多錯誤和安全漏洞，並讓程式碼更加臃腫，而且每次下提示詞都會翻攪許多程式碼，正確性很難驗證。
- 連結：https://www.youtube.com/watch?v=U46fJ2bJ-co

[5] C#之父 (Anders Hejlsberg)
- 連結更新於：2026年05月左右

- 連結：https://www.youtube.com/watch?v=CPrePbvbbic
